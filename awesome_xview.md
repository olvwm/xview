These are some of the other xview related repos that I found on Github:

- [IanDarwin/OpenLookCDROM](https://github.com/IanDarwin/OpenLookCDROM)
	The original OpenLook CDROM materials and documentation
- [MagnetarRocket/xview-openlook](https://github.com/MagnetarRocket/xview-openlook)
	An archival collection of xview code from various places on the web
- [ggodd/xview-64bit](https://github.com/ggodd/xview-64bit)
	A 64-bit port of XView based on code from Sourceforge (unmaintained
	as of 2021?)
- [retro-vault/open-look](https://github.com/retro-vault/open-look)
	Appears to be an attempt to use CMake instead of the original Imake to
	build a WM (olvw) based only on olgx (no more xview deps).
- ptribble's [version for Illumos](https://github.com/ptribble/xview-illumos)
